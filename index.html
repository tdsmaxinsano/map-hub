        Map Interaction

        Search any address or ZIP to place a location pin, click clinician markers for details, toggle home health or completed-service locations on only when needed, or enable Radius Mode and click the map to select clinicians within the chosen distance.

        TherapyBoss Launcher

        TherapyBoss blocks in-window embedding here, so this panel now works as a launch station for referrals and login flow.

        Use the launcher buttons to open TherapyBoss in a separate tab or window. That is the supported workflow for now.

        Recommended Workflow

        Keep the DependableCare map open here, then launch TherapyBoss beside it for referral review and login.

        1. Open TherapyBoss or Pending Referrals in a new tab.
        2. Log in there if prompted.
        3. Keep this dashboard open for clinician search, filtering, and profile review.
        4. Use the custom URL field when you want to jump to a specific TherapyBoss page.

        Data Import

        Upload a CSV or XLSX file, review the detected headers, map the fields you care about, preview the rows, and load them into Supabase.

        File

        Completed-services exports, pending-referral exports, and clinician master spreadsheets can all be mapped here. XLSX is now supported for the clinician roster too.

        Choose a CSV or XLSX file to begin.

        Field Mapping

        Header mapping is editable, so if TherapyBoss changes a column name you can still preview and stage the rows without touching code.

        Preview

        The preview shows the first staged rows after mapping, including clinician matches found from the current directory. Completed-services files also split semicolon-delimited clinician names into child assignment rows for upload.

        Uploads create TherapyBoss staging batches in Supabase. Completed-services imports preserve the parent service row, split multi-clinician assignments, and can geocode patient addresses into a reusable cache.
